> The interactive web portal for [grug-group420](https://github.com/grug-group420) — with a live CLI playground.

**🔗 [Visit the Portal →](https://grug-group420.github.io/portal)**

## What's Inside

- **Organization landing page** — overview of all grug-group420 projects
- **Live CLI playground** — try grugbot420 commands directly in the browser
- **Project showcase** — descriptions, links, and status for all repos

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Zero dependencies
- Zero build step
- Hosted via GitHub Pages

## Files

```
portal/
├── index.html    # Main page
├── styles.css    # Styles
└── app.js        # Playground logic
```

## Run Locally

```bash
git clone https://github.com/grug-group420/portal.git
cd portal
# Open index.html in your browser, or:
python3 -m http.server 8000
# Visit http://localhost:8000
```
