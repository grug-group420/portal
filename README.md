# 🪨 grug-group420 Portal

> "complexity very very bad" — grug

The official interactive portal and playground for [grug-group420](https://github.com/grug-group420).

🌐 **Live:** [grug-group420.github.io/portal](https://grug-group420.github.io/portal)

---

## ✨ Features

- **Live CLI Playground** — Try `grugbot420` commands directly in your browser
- **Project Showcase** — Browse all grug-group420 tools in one place
- **Zero Dependencies** — Pure HTML/CSS/JS, no frameworks, no build step
- **Run Locally** — One-click install instructions with copy buttons

## 🚀 Quick Start

```bash
# Clone and open
git clone https://github.com/grug-group420/portal.git
cd portal
open index.html
# or
python3 -m http.server 8000
```

No build step. No bundler. No node_modules. Just open the file. Grug way.
