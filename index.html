🧠 The Grug Way

We follow the sacred teachings of The Grug Brained Developer

✅ Grug Do

  • Write simple code
  • Ship early and often
  • Delete unnecessary code
  • Use boring technology
  • console.log for debugging
  • Read error messages

❌ Grug No Do

  • Over-engineer solutions
  • Add "just in case" features
  • Create deep abstractions
  • Chase new frameworks
  • Premature optimization
  • Clever one-liners
"Complexity very very bad. Say again: complexity very bad. Want to say again but then paragraph too long and complex." — The Grug Brained Developer

💻 Run Locally

Run grugbot420 on your machine with full CLI power

1

Install Bun

curl -fsSL https://bun.sh/install | bash
2

Clone & Run

git clone https://github.com/grug-group420/grugbot-server.git
cd grugbot-server
bun run serve
3

Open Browser

http://localhost:3420
